git clone <url> : create the respiratory to our local workplace
git status :to check whether git has initialized in ur corrent directory
git add <filename> : add the file "filename" to staging area(to perform commit and push)
git add . :add all the files in the directory to the staging area.
git commit -m "message" :to commit that it has been tracked by git
git push -u origin main : to push the commited file to the global workplace
git init : to initialize git in a folder
